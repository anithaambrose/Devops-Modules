
# Node.js - Node Package Manager 

  •	Used for Javascript Applications.
  
  •	Package Manager helps to manage libraries and dependencies, automate scripts, streamline development.
  
  •	Most popular in javascript is npm(node package manager).
  
  •	npm comes by default with Node.js installation
  
  •	It is used to install, update , remove packages(libraries)
  
  •	It maintains a file called package.json which tracks:
       *	project metadata
       *	Dependencies (libraries)  
       *	Scripts for running tasks.
