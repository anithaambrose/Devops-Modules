# Build Tools - Build & Package Managers in Software Development.

# What is a Build Tool ?

  •	A build tool, or software utility, is essential in contemporary software development 
  •	It Automates the process of building a project(converts Source code into executable binary .exe/.jar file)
  •	It automates the transformation of source code into an executable and deployable program.  
  
# Build Automation steps

  •	code compilation 
  •	tracks dependencies
  •	assessment - runs test cases locally
  •	Packages code 
  •	dependency management - deploys the code into local/ remote repo
