# CloudWatch

Amazon CloudWatch is a component of Amazon Web Services that provides monitoring for AWS resources and the customer applications running on the Amazon infrastructure.

Amazon Cloud Watch monitors your Amazon Web Services (AWS) resources and the applications you run on AWS in real time. 
You can use Cloud Watch to collect and track metrics, which are variables you can measure for your resources and applications

Use Cases
1. Monitor application performance - Visualize performance data, create alarms, and correlate data to understand and resolve the root cause of performance issues in your AWS resources.


2. Perform root cause analysis - Analyze metrics, logs, logs analytics, and user requests to speed up debugging and reduce overall mean time to resolution.
   
3.Optimize resources proactively - Automate resource planning and lower costs by setting actions to occur when thresholds are met based on your specifications or machine learning models.
